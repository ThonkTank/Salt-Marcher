# Repository File Inventory

## Purpose & Audience
This page provides a comprehensive, alphabetised list of every tracked file in the Salt Marcher repository for contributors who need a quick inventory while auditing assets or preparing releases.

## Directory Map
| Path | Description | Primary Docs |
| --- | --- | --- |
| `./` | Full repository tree. | [All Files](#all-files) |
| `wiki/` | Documentation pages that mirror the style guide template. | [Knowledge Base](./README.md) |

## Key Workflows
1. Review the file inventory before planning clean-up or archival tasks to confirm the current footprint.
2. Use your editor's search or Obsidian's quick switcher to jump to any filename listed under **All Files**.
3. Update this page after adding, removing, or renaming files so the index reflects the latest repository state.

## Linked Docs
- [Repository README](../README.md)
- [Documentation Hub](../DOCUMENTATION.md)
- [Style Guide](../style-guide.md)

## Standards & Conventions
- The inventory is generated with `find . -type f -not -path '*/.git/*' | sort` to ensure deterministic ordering.
- Paths are relative to the repository root and include spaces exactly as they appear on disk.
- Binary files (such as PDFs) are listed alongside Markdown and source files to provide a full audit trail.
