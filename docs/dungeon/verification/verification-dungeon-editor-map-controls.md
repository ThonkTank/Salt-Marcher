# Dungeon Editor Map, Projection, And Controls Matrix

This document owns route expectations for map catalog operations, startup responsiveness, camera controls, level/projection controls, overlay controls, and tool-family controls. The shared proof model, status vocabulary, gesture convention, route ownership, and completion criteria remain in [Dungeon Editor Tool Behavior Verification Catalog](verification-dungeon-editor-tool-behavior.md). Fixture definitions live in [Dungeon Editor Fixture Catalog](verification-dungeon-editor-fixtures.md).
