# Dungeon Editor Corridor Matrix

This document owns route expectations for corridor creation, anchor movement, route splitting, protected deletion, invalid-route rejection, and corridor point editing. The shared proof model, status vocabulary, gesture convention, route ownership, and completion criteria remain in [Dungeon Editor Tool Behavior Verification Catalog](verification-dungeon-editor-tool-behavior.md). Fixture definitions live in [Dungeon Editor Fixture Catalog](verification-dungeon-editor-fixtures.md).
