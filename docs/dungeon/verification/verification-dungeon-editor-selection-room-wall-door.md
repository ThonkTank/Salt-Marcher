# Dungeon Editor Selection, Room, Wall, And Door Matrix

This document owns route expectations for selection basics, room operations, wall operations, door operations, room narration, and room previews. The shared proof model, status vocabulary, gesture convention, route ownership, and completion criteria remain in [Dungeon Editor Tool Behavior Verification Catalog](verification-dungeon-editor-tool-behavior.md). Fixture definitions live in [Dungeon Editor Fixture Catalog](verification-dungeon-editor-fixtures.md).
