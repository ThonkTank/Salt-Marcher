# Dungeon Editor Stair And Transition Matrix

This document owns route expectations for stair creation/editing/deletion, stair validation, cross-level stair behavior, transition creation/deletion/editing, and transition linking. The shared proof model, status vocabulary, gesture convention, route ownership, and completion criteria remain in [Dungeon Editor Tool Behavior Verification Catalog](verification-dungeon-editor-tool-behavior.md). Fixture definitions live in [Dungeon Editor Fixture Catalog](verification-dungeon-editor-fixtures.md).
