# Session Planner Feature Docs

## Purpose

The `sessionplanner` feature owns the runtime planning workspace for one
adventure session. It combines the active party, saved encounter-plan budget
readouts, placed rests, and loot placeholders into one transient planning
surface.

It does not own party truth, encounter-plan persistence, creature truth, or
gold-economy rules.
